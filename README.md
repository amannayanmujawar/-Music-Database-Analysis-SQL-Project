# 🎵 Music Database SQL Analysis

## 📌 About

- 🟡 Rock listeners, top artists, longest songs
- 🔴 Spending per artist, popular genre per country, top customer per country

## 🛠️ Tools Used
- MySQL
- JOINs, Subqueries, CTEs, Window Functions

- `music_database.sql` — all queries included
